## Overview

**Organization**: CoderHeroes

- Mission: CoderHeroes establishes the next generation of innovators and disruptors by empowering opportunity youth and young adults to innovate community solutions through code

**Product:** Booking site for online classes

- A web application to aid in the cause of helping underprivileged youth in the Chicagoland area break into tech. Currently building an MVP booking site, and need to build out the rest of the site and functionality.
    - Currently using Square to facilitate bookings
- Parents buy tickets for either 1:1 or group classes
    - Like Eventbriteâ€”organizers make a scheduled event date with a given number of available seats, then parents would buy tickets/slots for each specific event
- Building a light LMS
    - System to manage **administration** and **attendance** of classes
